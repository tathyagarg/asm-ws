Run this Website

Run this website on your local machine (on the default port 57309) using the following commands

$ git clone https://github.com/tathyagarg/webserver.git $ cd webserver $ make

Run your own Website

Run your own website (must be written in vanilla HTML/CSS/JS, unfortunately) by simply putting it in the templates folder, and updating the templates/.endpoints file (See More). These commands show how to run a website with a single index.html file.

$ git clone https://github.com/tathyagarg/webserver.git $ cd webserver $ mv your-website templates $ echo "/ index.html\n" > templates/.endpoints $ make

Change your port

To change the port your server runs on, change the contents of the PORT file:

$ cd webserver $ echo "8080" > PORT $ make
